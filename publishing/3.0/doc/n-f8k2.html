%rendition-plus;

Rendition Markup Plus

Names a mix of the simplest of the typographic emphasis elements, as well as those for subscript and superscript, for use in models allowing data characters and this restricted mixture

Remarks

When this mix is enhanced in more permissive tag sets, best practice is for it to contain at least the typographic elements <sub> and <sup>.

Tag Set Maintenance Note: This parameter entity and the related parameter entity %just-rendition; have been put in place to restrict the amount of variability a person modifying the DTD through parameter entity redefinition can achieve. Some elements have been set as #PCDATA plus one parameter entity, while others have been set to #PCDATA plus the other, in an effort to allow designers to modify entire groups of elements, but not change similar models individually.
