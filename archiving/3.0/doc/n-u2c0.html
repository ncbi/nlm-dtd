<annotation>

Annotation in a Citation

Within a bibliographic reference (<element-citation> or <mixed-citation>), this element contains a commentary, summary, review, or similar non-bibliographic information concerning the cited work.

Remarks

While most citations simply provide the bibliographic information for a cited reference, a few describe the work, categorize the work, comment upon the nature or quality of the work, or summarize its findings. The <annotation> element is intended to hold such commentary.

Display Note: All of the other reference elements are inline elements. In contrast, an <annotation> is typically presented as a block element, with space before and after it.

Attribute

content-type Type of Content

Related Elements

This element is not related to the MathML <annotation> element, which is a mathematical term used only with equations.

Content Model

<!ELEMENT  annotation   ((%just-para.class;)+)                       >

Expanded Content Model

((p)+)

Description

Any one of:

  • <p> Paragraph, one or more

This element may be contained in:

<element-citation>, <mixed-citation>, <nlm-citation>, <product>, <related-article>, <related-object>
