initials

Initials of Given Names

The initial(s) for the given names of a person

Remarks

Use of this attribute is particularly useful in those cases where the person’s initials cannot be algorithmically determined, i.e., a program could not determine the initials of a given name due to the number, arrangement, or complexity of the name components.

Attribute Values

In Elements

<given-names>
<surname>
Value Meaning
Text, numbers, or special characters The initial letters of a person’s given names
Restriction: This attribute may be specified if the element is used.
