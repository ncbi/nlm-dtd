Accessibility Class Elements

Elements added to make it easier to process journal articles in ways that are more accessible to people and devices with special needs, for example, the visually handicapped. The <alt-text> element holds a short phrase description of an object such as a graphic, and the <long-desc> element contains a more complete description of the content or intent of an object.
