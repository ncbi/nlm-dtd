%inside-cell;

Inside Table Cell Elements

Elements that can be mixed with the data characters inside a table cell, that is, inside a <td> or <th> element in the XHTML table model

Remarks

This parameter entity will be used as the value of the table-model-specific parameter entities that name the content of table cells, for example, %FlowMix; for the XHTML table.
