underline-style

Appearance of the Underline

Indicates the style of underline to be used on display or in print, for example, “single”, “double”, or “dashed”

Usage

This attribute was added to preserve this information when a publisher supplies it. The use of the attribute doe not guarantee that the display will produce the formatting requested.

Attribute Values

In Element

<underline>
Value Meaning
Text, numbers, or special characters The style of underlining requested; for example, “double” would mean underlined by two rules, very close together.
Restriction: This attribute may be specified if the element is used.
