<attrib>

Attribution

A container element that holds the source, author, formal thanks, copyright material, or other information concerning the origin of an extract, display quote, poetry (<verse-group>), or similar element

Remarks

Formatting Note: The contents of this element are typically displayed on a separate line (or lines), following the material it concerns, thus inheriting that material’s margins.

Attributes

id Identifier
specific-use Specific Use

Content Model

<!ELEMENT  attrib       (#PCDATA %attrib-elements;)*                 >

Expanded Content Model

(#PCDATA | email | ext-link | uri | inline-supplementary-material | related-article | related-object | bold | italic | monospace | overline | roman | sc | strike | underline | alternatives | inline-graphic | chem-struct | inline-formula | abbrev | named-content | styled-content | fn | target | xref | sub | sup)*

Description

Any combination of:

  • Text, numbers, or special characters
  • All the address linking elements:
    • <email> Email Address
    • <ext-link> External Link
    • <uri> Uniform Resource Indicator (URI)
  • All the article linking elements:
    • <inline-supplementary-material> Inline Supplementary Material
    • <related-article> Related Article Information
    • <related-object> Related Object Information
  • All the emphasis elements:
    • <bold> Bold
    • <italic> Italic
    • <monospace> Monospace Text (Typewriter Text)
    • <overline> Overline
    • <roman> Roman
    • <sc> Small Caps
    • <strike> Strike Through
    • <underline> Underline
  • <alternatives> Alternatives For Processing
  • <inline-graphic> Graphic, Inline
  • <chem-struct> Chemical Structure (Display)
  • <inline-formula> Formula, Inline
  • All the inline elements used to surround “special” content:
    • <abbrev> Abbreviation or Acronym
    • <named-content> Named Special (Subject) Content
    • <styled-content> Styled Special (Subject) Content
  • All the simple link/cross-referencing elements:
    • <fn> Footnote
    • <target> Target of an Internal Link
    • <xref> X(cross) Reference
  • <sub> Subscript
  • <sup> Superscript

This element may be contained in:

<boxed-text>, <chem-struct-wrap>, <disp-formula>, <disp-quote>, <fig>, <graphic>, <media>, <preformat>, <statement>, <supplementary-material>, <table-wrap>, <table-wrap-foot>, <verse-group>
