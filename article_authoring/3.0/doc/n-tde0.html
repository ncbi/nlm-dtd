document-type

Document Type

Used to identify the type of document, for example, a chapter, glossary, review article, etc., that is (or contains) the target of the <related-object> element

Attribute Values

In Element

<related-object>
Value Meaning
Text, numbers, or special characters The type of related object, for example, “article” for a journal article
Restriction: This attribute may be specified if the element is used.
