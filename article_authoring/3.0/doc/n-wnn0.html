vol

Volume of Related Article

Volume of the journal in which a related article exists

Usage

The @vol attribute is used, along with the @journal-id, @issue, and @page attributes, to provide the metadata for the related article.

Attribute Values

In Element

<related-article>
Value Meaning
Text, numbers, or special characters The journal volume number for a related article, for example, “3” for the third volume, “125” for the one hundred and twenty-fifth, or “XV” for the fifteenth
Restriction: This attribute may be specified if the element is used.
